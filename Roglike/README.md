# SpartaCodingClub

현재 PERSONALPROJECT 디렉토리 핵심 js 파일은
server.js와 game.js 밖에 존재하지 않습니다...

Rogue like JAVASCRIPT 과제를 위한 새로운
디렉토리에는 Monster, Ueser를 위한 .js 파일과
키 입력에 따른 Action, 자주 사용될 기능을 묶어 놓을
디렉토리와 .js 파일로 만들어 볼 예정입니다.

현재 완성된 기능

1. 기본 공격
2. 확률에 따른 연속공격
3. 확률에 따른 방어
4. 방어 성공 시 확률에 따른 카운터 공격
5. 확률에 따른 도망가기
6. 유저 능력치 증가(정해진 범위 안에서 랜덤하게 능력치 증가)
7. 몬스터 HP와 최소, 최대 공격력이 정해진 범위 내에서 증가합니다

가 있습니다.

게임 진행 방식

1. 기본 공격 시 최소 데미지에서 최대 데미지의 범위 내 데미지를 몬스터에게 준다.
2. 몬스터에게 데미지를 입혔을 경우 데미지와 남은 몬스터의 HP를 출력한다.

3. 연속 공격 성공 시 기본 공격을 2번 한다.
4. 연속 공격 실패 시 실패했다는 문구를 출력한다.

5. 방어 성공 시 방어 성공 문구가 출력된다.
6. 방어에 성공했을 때 카운터도 성공하면 문구와 몬스터에게 기본 공격을 한다.
7. 방어 실패 시 실패했다는 문구가 출력된다.

8. 몬스터를 죽일 경우 최소, 최대 공격력이 증가한다.
9. 몬스터를 죽일 경우 확률로 발동되는 기능 1개가 일정 범위 내에서 증가한다.

10. 도망가기 성공 시 성공 문구가 출력 된다.
11. 도망가기 실패 시 실패 문구가 출력 된다.
12. 도망치기로 인해 라운드가 종료될 경우 능력치 상승은 이뤄지지 않는다.

13. 다음 라운드에 진입할 경우 유저의 체력을 50 회복 시켜준다.
14. 라운드 별 몬스터의 HP와 최소, 최대 공격력이 정해진 범위 내에서 랜덤하게 증가 합니다.
